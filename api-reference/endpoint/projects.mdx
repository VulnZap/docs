## List Projects

Get all projects in your organization.

### Request

```bash
curl https://api.vulnzap.com/v1/projects \
  -H "Authorization: Bearer vzap_abc123..."
```

### Response

```json
{
  "data": [
    {
      "project_id": "proj_xyz789",
      "name": "my-app",
      "repository_url": "https://github.com/example/my-app",
      "language": "typescript",
      "framework": "express",
      "risk_score": 34,
      "last_scan_at": "2025-10-19T14:32:11Z",
      "findings_count": {
        "critical": 0,
        "high": 2,
        "medium": 5,
        "low": 3
      },
      "created_at": "2025-01-15T10:22:30Z"
    }
  ]
}
```

## Create Project

Add a new project.

### Request

```bash
curl -X POST https://api.vulnzap.com/v1/projects \
  -H "Authorization: Bearer vzap_abc123..." \
  -H "Content-Type: application/json" \
  -d '{
    "name": "api-server",
    "repository_url": "https://github.com/example/api-server",
    "language": "python",
    "framework": "django"
  }'
```

### Response

```json
{
  "project_id": "proj_new123",
  "name": "api-server",
  "repository_url": "https://github.com/example/api-server",
  "created_at": "2025-10-19T15:20:11Z"
}
```

## Update Project

Modify project settings.

### Request

```bash
curl -X PATCH https://api.vulnzap.com/v1/projects/proj_xyz789 \
  -H "Authorization: Bearer vzap_abc123..." \
  -H "Content-Type: application/json" \
  -d '{
    "settings": {
      "auto_scan": true,
      "scan_schedule": "daily"
    }
  }'
```

## Delete Project

Remove a project (requires Admin/Owner).

### Request

```bash
curl -X DELETE https://api.vulnzap.com/v1/projects/proj_xyz789 \
  -H "Authorization: Bearer vzap_abc123..."
```

### Response

```json
{
  "project_id": "proj_xyz789",
  "deleted": true,
  "message": "Project and all associated data deleted"
}
```
