## Create Scan

Trigger a security scan for a project.

<ParamField path="project_id" type="string" required>
  Project identifier
</ParamField>

<ParamField path="files" type="array">
  Specific files to scan (optional, scans all if omitted)
</ParamField>

<ParamField path="incremental" type="boolean" default="false">
  Only scan changed files
</ParamField>

<ParamField path="min_severity" type="string" default="low">
  Minimum severity to report: `critical`, `high`, `medium`, `low`
</ParamField>

### Request

```bash
curl -X POST https://api.vulnzap.com/v1/scans \
  -H "Authorization: Bearer vzap_abc123..." \
  -H "Content-Type: application/json" \
  -d '{
    "project_id": "proj_xyz789",
    "files": ["src/auth.js", "src/db/queries.ts"],
    "incremental": false,
    "min_severity": "medium"
  }'
```

### Response

```json
{
  "scan_id": "scan_abc123",
  "status": "in_progress",
  "project_id": "proj_xyz789",
  "created_at": "2025-10-19T14:32:11Z",
  "estimated_completion": "2025-10-19T14:32:41Z"
}
```

## Get Scan Status

Retrieve scan results and status.

### Request

```bash
curl https://api.vulnzap.com/v1/scans/scan_abc123 \
  -H "Authorization: Bearer vzap_abc123..."
```

## List Scans

Get all scans for a project.

### Request

```bash
curl "https://api.vulnzap.com/v1/scans?project_id=proj_xyz789&page=1&per_page=20" \
  -H "Authorization: Bearer vzap_abc123..."
```

### Response

```json
{
  "data": [
    {
      "scan_id": "scan_abc123",
      "status": "completed",
      "created_at": "2025-10-19T14:32:11Z",
      "risk_score": 67
    }
  ],
  "pagination": {
    "page": 1,
    "per_page": 20,
    "total_pages": 5,
    "total_count": 87
  }
}
```

## Cancel Scan

Stop a running scan.

### Request

```bash
curl -X POST https://api.vulnzap.com/v1/scans/scan_abc123/cancel \
  -H "Authorization: Bearer vzap_abc123..."
```

### Response

```json
{
  "scan_id": "scan_abc123",
  "status": "cancelled",
  "message": "Scan cancelled successfully"
}
```
