## Installation

Install VulnZap globally:

```bash
npm i -g vulnzap
```

Or use via npx:

```bash
npx vulnzap init
```

## Authentication

Set your API key as an environment variable:

```bash
export VULNZAP_API_KEY="your_api_key_here"
```

Get your API key from [vulnzap.com/dashboard](https://vulnzap.com/dashboard) → Settings → API Keys

<Tip>
  Add `VULNZAP_API_KEY` to your shell profile (`~/.bashrc`, `~/.zshrc`) to persist across sessions
</Tip>

## Commands

### `vulnzap scan`

Scan your codebase for security vulnerabilities.

```bash
# Scan entire repository
vulnzap scan

# Scan specific files
vulnzap scan src/auth.js src/db/queries.ts

# Scan directory
vulnzap scan src/
```

### `vulnzap secure --ide <ide_name>`

Enable MCP integration for IDE security scanning.

**Supported IDEs:**

```bash
# For Cursor
vulnzap secure --ide cursor

# For Windsurf
vulnzap secure --ide windsurf
```

This command is used in MCP configuration files (`.cursor/mcp.json` or `.codeium/windsurf/mcp_config.json`).

## IDE Integration

### Cursor

Create `.cursor/mcp.json`:

```json
{
  "mcpServers": {
    "VulnZap": {
      "command": "vulnzap",
      "args": ["secure", "--ide", "cursor"],
      "env": {
        "VULNZAP_API_KEY": "your_api_key_here"
      }
    }
  }
}
```

### Windsurf

Create `.codeium/windsurf/mcp_config.json`:

```json
{
  "mcpServers": {
    "VulnZap": {
      "command": "vulnzap",
      "args": ["secure", "--ide", "windsurf"],
      "env": {
        "VULNZAP_API_KEY": "your_api_key_here"
      }
    }
  }
}
```

## CI/CD Integration

### GitHub Actions

```yaml
- name: Run VulnZap Scan
  run: vulnzap scan
  env:
    VULNZAP_API_KEY: ${{ secrets.VULNZAP_API_KEY }}
```

### GitLab CI

```yaml
vulnzap:
  script:
    - npm install -g vulnzap
    - vulnzap scan
  variables:
    VULNZAP_API_KEY: $VULNZAP_API_KEY
```
